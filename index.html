Betsy’s Stats Cookbook

A cookbook of applied-statistics code snippets accumulated while completing coursework for an applied-statistics degree. Each task is its own page, tagged with the course it came from and with cross-cutting concept tags (Model Fitting, Diagnostics, Visualization, …), so the same content is browsable by course and by concept — and fully searchable.

Use the category tags to the right to filter by course or by concept, or the search box (top right) to find a specific technique.
